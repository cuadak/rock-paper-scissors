Welcome to Rock Paper Scissors game

You will fight against a computer or bot.

This game using console in developer tools.

Click Ctrl + Shift + I combination button on your keyboard to start check the stats of the game.

Or you can use setting in right top corner => More tools => Developer tools after a window open click Console tab after Elements tab.

Once you start the game, u should finish it. To restart the game close the tab (Ctrl + W) then reopen closed tab (Ctrl + Shift + T)

To win the game you must reach 5 points.
